# 📚 Data Structures in C

A collection of Data Structure programs written in C language.
This repository contains implementations of Linked Lists, Stack, Queue, Expression Transformations, Browser History Simulation, and Music Playlist Management using basic data structure concepts.

---

# 🚀 Programs Included

## 1️⃣ Singly Linked List

File: `LInkedList.c`

Features:

* Insert at beginning
* Insert at end
* Insert at position
* Delete from beginning
* Delete from end
* Search element
* Display linked list

---

## 2️⃣ Circular Linked List

File: `CLL.c`

Features:

* Insert at beginning
* Insert at end
* Delete from beginning
* Delete from end
* Display circular linked list

---

## 3️⃣ Doubly Linked List

File: `DLL.c`

Features:

* Insert at beginning
* Insert at end
* Delete from beginning
* Delete from end
* Display forward
* Display backward

---

## 4️⃣ Stack Implementation

File: `stack.c`

Features:

* Push operation
* Pop operation
* Display stack
* Stack overflow & underflow handling

---

## 5️⃣ Queue Implementation

File: `Queue.c`

Features:

* Enqueue operation
* Dequeue operation
* Display queue
* Queue overflow & underflow handling

---

## 6️⃣ Expression Transformation

File: `Transformation.c`

Features:

* Infix to Postfix
* Postfix to Infix
* Prefix to Infix

---

## 7️⃣ Browser History Simulation

File: `BrowserHistory.c`

Features:

* Visit website
* Go back
* Display browser history

---

## 8️⃣ Music Playlist Management

File: `MusicPlaylist.c`

Features:

* Add songs
* Display playlist

---

# ▶️ How to Run

## Compile Program

```bash
gcc filename.c -o output
```

Example:

```bash
gcc stack.c -o stack
```

---

## Run Program

```bash
./output
```

Example:

```bash
./stack
```
